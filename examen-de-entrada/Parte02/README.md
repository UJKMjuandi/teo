# Guía de Compilación C++

## 1. ¿Cómo le proveemos al Sistema Operativo de un compilador C++?

**Windows:** Instalar MinGW-w64 o Visual Studio Community

**GNU/Linux:** 
```bash
sudo apt install build-essential
```

## 2. Comandos para compilar el programa

**Windows:**
```bash
g++ -o me.exe student.cpp me.cpp
```

**GNU/Linux:**
```bash
g++ -o me.out student.cpp me.cpp
```

## 3. ¿Cómo ejecutar el programa desde la terminal?

**Windows:**
```bash
me.exe
```

**GNU/Linux:**
```bash
./me.out
```
