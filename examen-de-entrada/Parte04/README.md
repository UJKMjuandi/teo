###Resolucion de la Parte 04
## 1. ¿Qué son los estándares de codificación?

Son reglas y convenciones para escribir código de manera consistente y legible. Incluyen formato, nomenclatura, estructura y mejores prácticas.

## 2. ¿Quién es considerado el padre de la Programación Orientada a Objetos?

**Alan Kay** es considerado el padre de la POO. Creó el término "programación orientada a objetos" y desarrolló el lenguaje Smalltalk.

## 3. ¿Qué diferencia hay entre editor de texto plano y editor de texto enriquecido?

**Editor de texto plano:** Guarda solo caracteres sin formato (Notepad, vim, nano)

**Editor de texto enriquecido:** Permite formato como negritas, colores, fuentes (Word, WordPad)

Para programación se usa texto plano.

## 4. ¿Cuál usar [TABS] o [ESPACIOS] para la indentación?

**Recomendación:** Usar **ESPACIOS** (2 o 4 espacios)

- Más consistente entre diferentes editores
- Evita problemas de visualización
- Estándar en la mayoría de guías de estilo
